# Sudais Portfolio

Professional product designer portfolio built with Next.js (App Router), TypeScript, and Tailwind CSS v4.

This project is intended to be deployed at:
- `sudais.skordlabs.com`

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Build and lint

```bash
pnpm lint
pnpm build
```

## Main routes

- `/` homepage
- `/about`
- `/case-studies`
- `/case-studies/[slug]`
- `/contact`

## Content

Case studies and focus areas are centralized in:
- `lib/portfolio-data.ts`
